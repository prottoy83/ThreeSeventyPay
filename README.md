# ThreeSeventyPay
A finance management and payment gateway system, made for 370 Project.

## Installation

Server Setup

```bash
  cd server
  npm install
```

Client Setup

```bash
   cd client
   npm install
```

## Run
 
Start MySQL server, then start server and client.

Starting server (Default port 5990)
```bash
   cd server
   node server.js
```

Start Client (Default Port: 5173)
```bash
   cd client
   npm run dev
```
