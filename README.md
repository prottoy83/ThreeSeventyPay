# ThreeSeventyPay
A finance management and payment gateway system, made for 370 Project.

## Installation

Server Setup

```bash
  cd server
  npm i express
```

## Run
 
Start MySQL server, then start server and client.

Starting server (Default port 5990)
```bash
   cd server
   node server.js
```
